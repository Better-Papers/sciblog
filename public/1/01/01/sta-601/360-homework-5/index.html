Hoff 5.2

mu_0 <- 75; sigma2_0 <- 100
kv_0 <- c(1, 2, 4, 8, 16, 32, 64)
n <- 16; iter <- 1000000

yA <- 75.2; sA <- 7.3
yB <- 77.5; sB <- 8.1

mc <- function(k) {
  k_n <- k + n; v_n <- k + n
  
  gen_sample <- function(ybar, s) {
    mu_n <- (k*mu_0 + n*ybar)/k_n
    s2_n <- 1/v_n * (k*sigma2_0 + (n-1)*s^2 + k*n*(ybar-mu_0)^2 / k_n)
    return(rnorm(iter, mean=mu_n, sd=sqrt(s2_n)))
  }
  
  return(mean(gen_sample(yA, sA) - gen_sample(yB, sB) < 0))
}

result <- sapply(kv_0, mc)
ggplot(data.frame(x=kv_0, y=result), aes(x=x, y=y)) + geom_point() +
  labs(x=TeX("$(\\kappa_0, \\nu_0)$"), y=TeX("Pr$(\\theta_A < \\theta_B \\, | \\, y_A, y_B)$"))

With low-informative priors (small \(\kappa_0, \nu_0\)), the probability that \(\theta_A < \theta_B\) is not high to begin with \((0.58)\). With stronger priors \(\mu_0=75\approx\bar{x}_A\), the probability sinks lower as there’s a stronger belief in the prior.

Jeffrey’s Prior

\[ \begin{align*} p(\theta, \sigma^2) &= \left(\sigma^2\right)^{-3/2} \\ Y_1, \dots, Y_n &\sim \mathcal{N}(\theta, \sigma^2) \\~\\ p(\theta, \sigma^2 \mid Y_1, \dots, Y_n) &\propto p(\theta, \sigma^2)\ p(Y_1, \dots, Y_n \mid \theta, \sigma^2)\\ &= (\sigma^2)^{-3/2}\ \left[(\sigma^2)^{-n/2} \exp \left\{ -\frac{1}{2\sigma^2} \sum_{i=1}^n \left(y_i-\theta\right)^2 \right\}\right] \\ &= (\sigma^2)^{-(n+3)/2} \exp \left\{ -\frac{1}{2\sigma^2} \sum_{i=1}^n \left((y_i-\bar{y})+(\bar{y}-\theta)\right)^2 \right\} \\ &= (\sigma^2)^{-(n+3)/2} \exp \left\{ -\frac{1}{2\sigma^2} \sum_{i=1}^n \left((y_i-\bar{y})^2+2(y_i-\bar{y})(\bar{y}-\theta)+(\bar{y}-\theta)^2\right) \right\} \\ &= (\sigma^2)^{-(n+3)/2} \exp \left\{ -\frac{1}{2\sigma^2} \left[\sum_{i=1}^n (y_i-\bar{y})^2+2(\bar{y}-\theta)\left(\sum y_i-n\bar{y}\right)+n(\bar{y}-\theta)^2 \right]\right\} \\ &= (\sigma^2)^{-(n+3)/2} \exp \left\{ -\frac{1}{2\sigma^2} \left[\sum_{i=1}^n (y_i-\bar{y})^2+0+n(\bar{y}-\theta)^2 \right]\right\} \\ &= (\sigma^2)^{-(n+3)/2} \exp \left\{ -\frac{1}{2\sigma^2} \left[(n-1)s^2+n(\bar{y}-\theta)^2\right]\right\} \\ &= \operatorname{Normal-InvGamma}\left(\theta, n, \frac{n+1}{2}, \frac{(n-1)s^2}{2}\right) \end{align*} \] where \[ s^2 = \frac{1}{n-1} \sum_{i=1}^n (y_i-\bar{y})^2 \\ \operatorname{Normal-InvGamma}\left(\mu, \lambda, \alpha, \beta \right) \]

When \(n=0\), the posterior is improper as the normal-inverse-gamma distribution is only valid for parameters \(\lambda, \alpha, \beta > 0\). With \(n=1\), however, the posterior is proper.
